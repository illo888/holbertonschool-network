# Networking Basics #0

This project covers the fundamental concepts of computer networking including the OSI model, types of networks, MAC and IP addresses, and TCP/UDP protocols.

## Tasks

### 0. OSI model
Answer questions about the OSI model.

### 1. Types of network
Answer questions about LAN, WAN, and the Internet.

### 2. MAC and IP address
Answer questions about MAC addresses and IP addresses.

### 3. UDP and TCP
Answer questions about UDP and TCP protocols.

### 4. TCP and UDP ports
Write a Bash script that displays listening ports with PID and program names.

### 5. Is the host on the network
Write a Bash script that pings an IP address passed as an argument 5 times.
